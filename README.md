# صيدلي - Seidaly Dashboard MVP

مساعدك الدوائي الذكي وجدول أدوية عائلتك.

## التشغيل المحلي

**المتطلبات:** Node.js

1. تثبيت المكتبات:
   `npm install`
2. ضع مفتاح `GEMINI_API_KEY` في ملف [.env](.env)
3. تشغيل التطبيق:
   `npm run dev`
4. للوصول من الموبايل: تأكد أن هاتفك على نفس شبكة الـ Wi-Fi وافتح الرابط المعروض في الطرفية (Terminal).
